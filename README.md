# Book-Bank-Management-System

## About
This is a simple Book Bank System. The system helps manage books by allowing users to add, view, edit, and delete book records.

## Features
- Add books
- View books
- Edit book details
- Delete books

## Technologies Used
- HTML
- CSS
- PHP
- MySQL
- XAMPP

## How to Run
1. Start Apache and MySQL in XAMPP.
2. Copy the project folder into the `htdocs` folder.
3. Create a database named `book_bank`.
4. Import the SQL file.
5. Open the project in your browser using:
   `http://localhost/book_bank/`
